# Moments Wall E2E Tests — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Full end-to-end Playwright test suite that creates a real event, uploads a photo through the public frontend, and verifies every Moments Wall feature in the dashboard.

**Architecture:** Single file `tests/e2e/moments.spec.ts` using Playwright's `test.describe.serial` for sequential execution and shared state. Upload is done via `setInputFiles` on the hidden file input of `localhost:4321/events/{identifier}/upload`. Dashboard runs on `localhost:3000`. Auth reuses existing `tests/e2e/.auth/session.json`. A small JPEG fixture at `tests/e2e/fixtures/test-photo.jpg` is created with a Node script.

**Tech Stack:** Playwright 1.58, TypeScript, real backend + real public frontend (both must be running locally)

```js
// scripts/create-test-photo.mjs
// Creates a minimal 10x10 white JPEG for use in E2E tests.
// Run once: node scripts/create-test-photo.mjs
import sharp from 'sharp'
import { join, dirname } from 'path'
import { fileURLToPath } from 'url'
import { mkdirSync } from 'fs'

const __dirname = dirname(fileURLToPath(import.meta.url))
const out = join(__dirname, '../tests/e2e/fixtures/test-photo.jpg')
mkdirSync(join(__dirname, '../tests/e2e/fixtures'), { recursive: true })

await sharp({
  create: { width: 10, height: 10, channels: 3, background: { r: 200, g: 200, b: 200 } }
}).jpeg().toFile(out)

console.log('✓ tests/e2e/fixtures/test-photo.jpg created')
```

### Full spec content

```typescript
/**
 * moments.spec.ts — Full E2E for the Moments Wall
 *
 * Prerequisites (must be running):
 *   - Dashboard:       npm run dev          → http://localhost:3000
 *   - Public frontend: npm run dev          → http://localhost:4321  (cafetton-casero)
 *   - Backend:         running at localhost:8080
 *   - .env.local:      TEST_EMAIL + TEST_PASSWORD
 *
 * Flow:
 *   1. Create test event
 *   2. Enable QR shared uploads
 *   3. Upload photo as guest via public frontend
 *   4. Verify moment appears in dashboard (pending → approved)
 *   5. Test Moments Wall features: Fotos tab, multi-select, lightbox,
 *      ZIP dropdown, Por hora toggle
 *   6. Cleanup: delete event
 */

import { test, expect, type Page } from '@playwright/test'
import path from 'path'

test.use({ storageState: 'tests/e2e/.auth/session.json' })

// Shared state across serial tests
let eventId = ''
let eventIdentifier = ''
const FIXTURE_PHOTO = path.join(__dirname, 'fixtures/test-photo.jpg')
const PUBLIC_URL = 'http://localhost:4321'

// ─── Helpers ────────────────────────────────────────────────────────────────

/** Navigate to the test event detail page and wait for it to load */
async function goToEvent(page: Page) {
  await page.goto(`/events/${eventId}`)
  await page.waitForLoadState('networkidle')
}

/** Navigate to events list, find and click the test event link */
async function goToEventFromList(page: Page) {
  await page.goto('/events')
  await page.waitForLoadState('networkidle')
  await page.getByRole('link', { name: new RegExp('E2E Test Moments', 'i') }).first().click()
  await page.waitForLoadState('networkidle')
}

// ─── Tests ──────────────────────────────────────────────────────────────────

test.describe.serial('Moments Wall — flujo completo', () => {

  // ── 1. Create test event ─────────────────────────────────────────────────

  test('crear evento de prueba', async ({ page }) => {
    await page.goto('/events')
    await page.waitForLoadState('networkidle')

    await page.getByRole('button', { name: 'Crear evento' }).click()
    await expect(page.getByRole('dialog')).toBeVisible()

    // Fill form
    await page.getByLabel('Nombre del evento').fill('E2E Test Moments')
    // event_date_time: datetime-local input (YYYY-MM-DDTHH:mm)
    await page.locator('input[type="datetime-local"]').fill('2026-12-31T20:00')
    // Timezone has a default (America/Mexico_City) — no change needed
    // Address is optional in schema — skip
    await page.getByTestId('submit-event-form').click()

    // After creation, router.push navigates to /events/{id}
    await page.waitForURL(/\/events\/[a-z0-9-]{36}/, { timeout: 15_000 })
    await page.waitForLoadState('networkidle')

    // Capture the event ID from URL
    const url = page.url()
    const match = url.match(/\/events\/([a-z0-9-]{36})/)
    expect(match).not.toBeNull()
    eventId = match![1]
    expect(eventId.length).toBeGreaterThan(0)

    // Capture identifier from the page (shown in the public links)
    // The "Ver en eventiapp" link contains the identifier
    const previewLink = page.locator('a[href*="/e/"]').first()
    if (await previewLink.isVisible()) {
      const href = await previewLink.getAttribute('href') ?? ''
      const idMatch = href.match(/\/e\/([^/?]+)/)
      if (idMatch) eventIdentifier = idMatch[1]
    }

    // Fallback: try to get from upload link
    if (!eventIdentifier) {
      const uploadLink = page.locator('a[href*="/events/"][href*="/upload"]').first()
      if (await uploadLink.isVisible()) {
        const href = await uploadLink.getAttribute('href') ?? ''
        const idMatch = href.match(/\/events\/([^/]+)\/upload/)
        if (idMatch) eventIdentifier = idMatch[1]
      }
    }

    expect(eventIdentifier.length).toBeGreaterThan(0)

    // Event heading visible
    await expect(page.getByRole('heading', { name: 'E2E Test Moments' })).toBeVisible()
  })

  // ── 2. Enable QR shared uploads ──────────────────────────────────────────

  test('habilitar subida QR compartida', async ({ page }) => {
    await goToEvent(page)

    // Find and click the "Habilitar subida QR" button in MomentsWall toolbar
    const qrBtn = page.getByTitle(/habilitar subida qr|subida qr activa/i).first()
    const isAlreadyEnabled = (await qrBtn.getAttribute('title') ?? '').toLowerCase().includes('activa')

    if (!isAlreadyEnabled) {
      await qrBtn.click()
      // Wait for state change
      await expect(page.getByTitle(/subida qr activa/i)).toBeVisible({ timeout: 8_000 })
    }

    // Verify QR upload is now enabled
    await expect(page.getByTitle(/subida qr activa/i)).toBeVisible()
  })

  // ── 3. Upload photo as guest via public frontend ──────────────────────────

  test('subir foto como invitado (app pública)', async ({ browser }) => {
    // Use a fresh browser context (no dashboard auth) to simulate a guest
    const guestContext = await browser.newContext()
    const guestPage = await guestContext.newPage()

    const uploadUrl = `${PUBLIC_URL}/events/${eventIdentifier}/upload`
    await guestPage.goto(uploadUrl)
    await guestPage.waitForLoadState('networkidle')

    // Verify upload page loaded (not "uploads not enabled" screen)
    await expect(guestPage.getByText(/Seleccionar de galería|Agrega|comparte/i)).toBeVisible({ timeout: 10_000 })

    // Set the file on the hidden file input
    const fileInput = guestPage.locator('input[type="file"]').first()
    await fileInput.setInputFiles(FIXTURE_PHOTO)

    // Wait for preview to appear (file name or thumbnail visible)
    await expect(guestPage.getByText(/test-photo\.jpg|\.jpg/i).or(
      guestPage.locator('.aspect-square img, canvas').first()
    )).toBeVisible({ timeout: 8_000 })

    // Click the upload/submit button
    const submitBtn = guestPage.getByRole('button', { name: /Compartir momento|Compartir \d+ momento/i })
    await expect(submitBtn).toBeVisible({ timeout: 5_000 })
    await submitBtn.click()

    // Wait for success screen
    await expect(guestPage.getByText(/¡Momento compartido!/i)).toBeVisible({ timeout: 30_000 })

    await guestContext.close()
  })

  // ── 4. Verify moment appears as pending in dashboard ─────────────────────

  test('el momento aparece en Pendientes', async ({ page }) => {
    await goToEvent(page)

    // Wait for the moments to load — poll every 2s for up to 30s
    // (15s auto-refresh + processing time)
    await expect(async () => {
      await page.reload()
      await page.waitForLoadState('networkidle')
      await expect(
        page.getByRole('tab', { name: /Pendientes/ })
          .filter({ hasText: /[1-9]/ })
      ).toBeVisible()
    }).toPass({ timeout: 60_000, intervals: [5_000] })

    // Click Pendientes tab
    await page.getByRole('tab', { name: /Pendientes/ }).click()

    // At least one moment card visible
    const card = page.locator('.group.aspect-square, [class*="aspect-square"]').first()
    await expect(card).toBeVisible({ timeout: 5_000 })
  })

  // ── 5. Approve the moment ─────────────────────────────────────────────────

  test('aprobar el momento', async ({ page }) => {
    await goToEvent(page)

    // Click Pendientes tab
    await page.getByRole('tab', { name: /Pendientes/ }).click()
    await page.waitForTimeout(500)

    // Hover the first card to reveal action bar, then click Aprobar
    const card = page.locator('.group.aspect-square').first()
    await card.hover()
    await page.getByRole('button', { name: /Aprobar/i }).first().click()

    // Wait for success toast
    await expect(page.getByText(/Momento aprobado/i)).toBeVisible({ timeout: 8_000 })

    // Approved tab now shows count ≥ 1
    await expect(
      page.getByRole('tab', { name: /Aprobados/ }).filter({ hasText: /[1-9]/ })
    ).toBeVisible({ timeout: 5_000 })
  })

  // ── 6. Approved moment appears in Fotos tab ───────────────────────────────

  test('el momento aprobado aparece en la tab Fotos', async ({ page }) => {
    await goToEvent(page)

    // Fotos tab only appears when photoCount > 0
    await expect(page.getByRole('tab', { name: /Fotos/ })).toBeVisible({ timeout: 5_000 })
    await page.getByRole('tab', { name: /Fotos/ }).click()

    const card = page.locator('.group.aspect-square').first()
    await expect(card).toBeVisible({ timeout: 5_000 })
  })

  // ── 7. Multi-select ───────────────────────────────────────────────────────

  test('multi-select: seleccionar momento activa botones de bulk action', async ({ page }) => {
    await goToEvent(page)
    await page.getByRole('tab', { name: /Aprobados/ }).click()
    await page.waitForTimeout(300)

    // Enter select mode
    await page.getByTitle(/seleccionar momentos/i).click()
    await expect(page.getByText(/Seleccionar todo/i)).toBeVisible()

    // Select all via checkbox
    await page.getByRole('checkbox').click()

    // Bulk action buttons appear
    await expect(page.getByText(/Aprobar selección|Eliminar selección/i)).toBeVisible({ timeout: 3_000 })

    // Exit select mode
    await page.getByTitle(/cancelar selección/i).click()
  })

  // ── 8. Lightbox ───────────────────────────────────────────────────────────

  test('lightbox: abrir y navegar momento', async ({ page }) => {
    await goToEvent(page)
    await page.getByRole('tab', { name: /Aprobados/ }).click()
    await page.waitForTimeout(300)

    // Click the first card to open lightbox
    const card = page.locator('.group.aspect-square').first()
    await card.click()

    // Lightbox overlay visible (fixed full-screen element)
    await expect(page.locator('.fixed.inset-0').last()).toBeVisible({ timeout: 5_000 })

    // Close button visible
    await expect(page.getByRole('button', { name: /Cerrar/i })).toBeVisible()

    // Close lightbox
    await page.keyboard.press('Escape')
    await expect(page.locator('.fixed.inset-0').last()).not.toBeVisible({ timeout: 3_000 })
  })

  // ── 9. ZIP split dropdown ─────────────────────────────────────────────────

  test('ZIP split: dropdown muestra opciones Solo fotos y Solo vídeos', async ({ page }) => {
    await goToEvent(page)

    // The chevron button next to ZIP
    const zipOptions = page.getByTitle(/opciones de descarga/i)
    await expect(zipOptions).toBeVisible()
    await zipOptions.click()

    await expect(page.getByText('Solo fotos')).toBeVisible({ timeout: 3_000 })
    await expect(page.getByText('Solo vídeos')).toBeVisible()

    // Close by pressing Escape or clicking elsewhere
    await page.keyboard.press('Escape')
  })

  // ── 10. Por hora toggle ───────────────────────────────────────────────────

  test('Por hora: agrupar momentos por tiempo', async ({ page }) => {
    await goToEvent(page)
    await page.getByRole('tab', { name: /Aprobados/ }).click()
    await page.waitForTimeout(300)

    // Click the Por hora toggle
    await page.getByTitle(/agrupar por hora/i).click()

    // A time bucket label should appear (format HH:MM)
    await expect(page.locator('text=/\\d{2}:\\d{2}/').first()).toBeVisible({ timeout: 5_000 })

    // Toggle back off
    await page.getByTitle(/agrupar por hora/i).click()
  })

  // ── 11. Cleanup ───────────────────────────────────────────────────────────

  test('limpiar: eliminar evento de prueba', async ({ request }) => {
    // Use API request to delete — avoids needing the dashboard UI for cleanup
    // The existing api client uses Cognito tokens; here we rely on the session
    // cookie already established via auth setup.
    // Delete via the backend REST API
    if (!eventId) return

    // Navigate to events list and delete via UI (safer than direct API call
    // since we don't have the Cognito token directly in Playwright request)
    // We skip full UI deletion and just mark it inactive via API
    // NOTE: If the backend supports DELETE /events/:id, use that.
    // Otherwise skip — the test event name "E2E Test Moments" makes it easy
    // to identify and manually clean up.
    console.log(`[cleanup] Test event ID: ${eventId} — delete manually if needed`)
  })
})
```

### Step 2: Fix issues as they arise

**Known likely issues and fixes:**

**Issue A: `datetime-local` input not found**
The label is "Fecha y hora" but the input type might differ. Fix:
```typescript
// Instead of:
await page.locator('input[type="datetime-local"]').fill('2026-12-31T20:00')
// Try:
await page.getByLabel('Fecha y hora').fill('2026-12-31T20:00')
```

**Issue B: Event identifier not captured from links**
If the "Ver en eventiapp" link isn't visible immediately, try getting identifier from the page URL by loading event config:
```typescript
// Alternative: get from the upload URL shown in the toolbar QR button tooltip
// or from the wall share modal
// Or: fetch from backend API using the event ID
const resp = await page.request.get(`http://localhost:8080/api/events/${eventId}`)
const data = await resp.json()
eventIdentifier = data.data?.identifier ?? data.identifier
```

**Issue C: Upload page shows "uploads not enabled"**
The shared uploads weren't enabled fast enough. Add a retry in the upload step:
```typescript
// Wait for the upload page to show the drop zone (not the disabled state)
await expect(
  guestPage.getByText(/Seleccionar de galería/i)
).toBeVisible({ timeout: 15_000 })
```

**Issue D: Card hover doesn't reveal action bar on desktop**
The action bar is `sm:opacity-0 sm:group-hover:opacity-100`. Playwright hover should work. If not:
```typescript
// Force hover
await card.hover({ force: true })
await page.waitForTimeout(300) // allow CSS transition
await page.getByRole('button', { name: /Aprobar/i }).first().click({ force: true })
```

**Issue E: Lightbox close with Escape doesn't work**
The Escape handler checks `if (e.key === 'Escape') onClose()`. If keyboard events don't propagate:
```typescript
// Click the close button instead
await page.getByRole('button', { name: /Cerrar/i }).click()
// Or use aria-label
await page.getByLabel('Cerrar').click()
```

**Issue F: Moment still processing after upload**
Lambda processing may delay. Increase poll timeout in test 4:
```typescript
// Already using toPass with 60s timeout — if still failing, increase to 90s
{ timeout: 90_000, intervals: [5_000] }
```

**Issue G: `.group.aspect-square` selector not matching**
The card outer div has `className="relative rounded-xl overflow-hidden bg-zinc-900 group aspect-square"`. Try:
```typescript
// More specific:
page.locator('div.group.aspect-square')
// Or by role — cards don't have explicit roles, use nth:
page.locator('[class*="aspect-square"][class*="group"]').first()
```

## Important notes

- Run only `--project=chromium` (not mobile) for moments tests — file input + hover interactions are more reliable on desktop
- The `test.describe.serial` ensures sequential order and shared `eventId`/`eventIdentifier` state
- If the backend doesn't have a DELETE /events endpoint, the cleanup test just logs the ID — that's fine
- The test event name "E2E Test Moments" makes it easy to identify and manually clean if needed
- Never use `page.waitForTimeout` as the primary wait — only as a stabilization pause after state changes
